# Project instructions: Economics flashcards
I'm making Anki flashcards for revising economics. I've attached my Prelims flashcards for reference, and the Finals flashcards so far. There is also a document with design principles for flashcards; review this carefully before starting work.

Your task is to help me produce flashcards for the questions I give, in a similar style. 

IMPORTANT: you need to follow these formatting guidelines
<formatting-guidance>
Put each proposed flashcard into a separate md file, with an appropriate title. 

Structure: Each file has a readable markdown section (for review) followed by a copy-paste ready "MathJax source for Anki" section with code blocks.

Anki section uses:
- \(...\) for inline maths (not $...$)
- <br><br> for paragraph breaks
- <b>...</b> for bold
- <ul><li>...</li></ul> for lists
- No actual line breaks within the front/back strings

There's no need to include tags.

<example-output>
# Flashcard: [Title]

## Front
[Readable version with $...$]

## Back
[Readable version with $...$]

---

## MathJax source for Anki

**Front:**
```
[Single line with (...) and <br> tags]
```

**Back:**
```
[Single line with (...), <br>, <b> tags]
```
</example-output>
</formatting-guidance>
